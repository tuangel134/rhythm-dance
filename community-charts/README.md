# Charts de la comunidad

Esta carpeta almacena los **mapeos de notas (charts)** que la comunidad de Rhythm
Dance comparte. **Nunca contiene audio**: solo notas y metadatos.

- `index.json` — índice de todos los charts publicados (para buscar sin descargar todo).
- `charts/<fingerprint>/<packageId>.json` — un chart por archivo, agrupado por canción.

Cada usuario debe poseer su propio archivo de audio. El chart se empareja con tu
canción local mediante una "huella" calculada de título + artista + duración + BPM,
no del contenido sonoro. Respeta los derechos de autor: no subas audio.

Los charts se publican desde el propio juego (pestaña **Comunidad**) con un token
de GitHub, o por Pull Request para revisión.
